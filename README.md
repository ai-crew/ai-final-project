### Getting Started
* Installing dependencies:
```
pip install -r requirements.txt
```

* Running the app:
```
python app.py
```

* Code clean up:
```
black app.py
```

* Saving new dependencies:
```
pipreqs --force
```
Note: pipreqs only saves project dependencies.
