### Getting Started
* Installing dependencies:
```
pip install -r requirements.txt
```

* Running the app:
```
python app.py
```

* Code clean up:
```
black app.py
```

* Saving new dependencies:
```
pip install pipreqs   #if not already installed
pipreqs --force
```
Note: if pipreqs doesn't work try the following:
```
pip freeze > requirements.txt
```
