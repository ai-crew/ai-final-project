### Getting Started
* Installing dependencies:
```
pip install -r requirements.txt
```

* Running the app:
```
python app.py
```

* Code clean up:
```
black app.py
```

* Adding new dependencies:
```
pipreqs --force
```
(if that doesn't work try the following)
```
pip freeze > requirements.txt
```
